# Day 50: Capstone Project - Building Your LLM Application - Further Reading

This final day synthesizes all concepts into a comprehensive capstone project demonstrating your mastery of large language models. Here are resources for building production-grade applications.

## References

1. **Building LLM Applications: The Definitive Guide**
   - [LangChain Cookbook](https://github.com/langchain-ai/langchain/blob/master/docs/docs_structure.md)
   - Collection of production patterns and best practices for building with LLMs.

2. **LLM Application Architecture**
   - [Building Durable LLM Applications](https://www.ankush.dev/p/building-robust-llm-apps)
   - Comprehensive guide to production LLM system design and architecture.

3. **Evaluation Frameworks for LLM Applications**
   - [Ragas: Evaluation for RAG](https://github.com/explodinggradients/ragas)
   - Framework for evaluating RAG and LLM-based applications systematically.

4. **Observability and Monitoring**
   - [Langfuse: Observability for LLM Applications](https://langfuse.com/)
   - Platform for monitoring, debugging, and improving LLM applications in production.

5. **Best Practices for LLM Engineering**
   - [Chips and Salsa: Lessons from MLOps](https://www.databricks.com/blog/2023/10/05/improve-your-llms-validity-reducing-hallucinations.html)
   - Industry best practices and lessons from deploying LLM systems at scale.

**Tip:** Study all references to build a comprehensive, production-ready LLM application demonstrating your full skillset from Phases 1-5.
