# Day 10: Project 1 - MNIST Classification with Custom NN - Further Reading

This day synthesizes all Phase 1 concepts into a comprehensive project. Here are resources for best practices in building, evaluating, and deploying neural networks.

## References

1. **Deep Learning Best Practices: A Practical Guide**
   - [Jeremy Howard's Fast.ai Practical Deep Learning](https://course.fast.ai/)
   - Top-down approach covering practical best practices for building effective models from data collection to deployment.

2. **PyTorch Lightning: Structuring Deep Learning Code**
   - [PyTorch Lightning Documentation](https://pytorch-lightning.readthedocs.io/)
   - Framework for organizing complex training pipelines with best practices for reproducibility and scalability.

3. **Model Interpretability: Grad-CAM and Visualization**
   - [Grad-CAM Paper](https://arxiv.org/abs/1610.02055)
   - Guide to visualizing which parts of an image the model attends to, enabling better model understanding and debugging.

4. **Deploying Deep Learning Models**
   - [ONNX: Open Neural Network Exchange](https://onnx.ai/)
   - Standard format for exporting neural network models across frameworks and deploying to production efficiently.

5. **Building Production ML Systems**
   - [Google MLOps.community](https://mlops.community/)
   - Resources on operationalizing machine learning models including monitoring, versioning, and continuous improvement.

---

**Tip:** Start with reference #1 for practical project structure, #2 for code organization, and #3 for model analysis and interpretation.
