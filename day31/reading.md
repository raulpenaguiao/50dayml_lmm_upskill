# Day 31: Introduction to Hugging Face Transformers - Further Reading

This day introduces the Hugging Face Transformers library, the most widely-used toolkit for working with pretrained language models, providing easy access to state-of-the-art models.

## References

1. **Hugging Face Transformers Library Documentation**
   - [Hugging Face: Transformers Docs](https://huggingface.co/docs/transformers/)
   - Official comprehensive documentation covering models, tokenizers, pipelines, and best practices.

2. **Hugging Face Course: NLP with Transformers**
   - [HF Course](https://huggingface.co/course/)
   - Free comprehensive course covering the Transformers library with hands-on coding exercises.

3. **The Hugging Face Model Hub**
   - [Model Hub: huggingface.co/models](https://huggingface.co/models)
   - Browse and discover thousands of pretrained models, datasets, and community implementations.

4. **Pipelines: High-Level API**
   - [HF Pipelines Guide](https://huggingface.co/docs/transformers/main_classes/pipelines)
   - Simple high-level API for common NLP tasks requiring minimal code.

5. **Tokenizers Library**
   - [Hugging Face Tokenizers](https://huggingface.co/docs/tokenizers/)
   - Fast tokenizer implementations for modern LLMs, can be used standalone or with Transformers.

---

**Tip:** Start with reference #2 for comprehensive learning, #1 for reference, #4 for quick tasks, and #3 to explore available models.
