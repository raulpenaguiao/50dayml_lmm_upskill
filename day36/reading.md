# Day 36: Model Deployment with FastAPI - Further Reading

This day covers deploying machine learning models using FastAPI, a modern framework for building high-performance REST APIs for model inference.

## References

1. **FastAPI Official Documentation**
   - [FastAPI Docs](https://fastapi.tiangolo.com/)
   - Complete official documentation with tutorials, examples, and API reference.

2. **Building Machine Learning APIs with FastAPI**
   - [Real Python: FastAPI and ML Models](https://realpython.com/fastapi/)
   - Practical guide to building and deploying ML models with FastAPI.

3. **Serving PyTorch Models with FastAPI**
   - [PyTorch Serving Guide](https://pytorch.org/serve/)
   - Official PyTorch guide for serving models in production environments.

4. **API Design Best Practices**
   - [REST API Design Guidelines](https://restfulapi.net/)
   - Best practices for designing clean, scalable REST APIs.

5. **Testing and Documentation**
   - [pytest: Testing Framework](https://docs.pytest.org/)
   - Framework for comprehensive testing of APIs and model endpoints.
