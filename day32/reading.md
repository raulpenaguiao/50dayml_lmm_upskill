# Day 32: Fine-Tuning BERT for Text Classification - Further Reading

This day covers fine-tuning pretrained BERT models on custom text classification tasks using the Hugging Face Transformers library and Trainer API.

1. **Fine-tuning BERT for Text Classification**
   - [HF Fine-tuning Guide](https://huggingface.co/docs/transformers/training)
   - Official guide to fine-tuning BERT and other models with the Trainer API.

2. **The Trainer and TrainingArguments**

3. **Hugging Face Datasets Library**
