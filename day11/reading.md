# Day 11: Introduction to Natural Language Processing - Further Reading

This day introduces Natural Language Processing (NLP), a field dealing with the interaction between computers and human language. Understanding text preprocessing is the foundation for all downstream NLP tasks.

## References

1. **NLTK Book: Natural Language Processing with Python**
   - [NLTK Book - Processing Raw Text](https://www.nltk.org/book/)
   - Comprehensive guide to NLP fundamentals with practical examples using the Natural Language Toolkit, covering tokenization, stemming, and lemmatization.

2. **SpaCy: Advanced NLP Library**
   - [SpaCy Documentation](https://spacy.io/)
   - Modern, efficient NLP library with pre-trained pipelines for tokenization, POS tagging, and text preprocessing in production settings.

3. **Understanding Tokenization**
   - [Hugging Face: Tokenization Guide](https://huggingface.co/docs/transformers/tokenization_summary)
   - Comprehensive guide to different tokenization strategies (word-level, subword, character-level) essential for understanding modern NLP.

4. **Text Preprocessing Best Practices**
   - [Towards Data Science: Text Cleaning Guide](https://towardsdatascience.com/nlp-text-preprocessing-a-practical-guide-and-template-d80874676e45)
   - Practical guide to text cleaning, normalization, and handling special cases in real-world NLP projects.

5. **IMDb and AG News Datasets**
   - [Hugging Face Datasets: Text Classification](https://huggingface.co/datasets?task=text-classification)
   - Easy access to popular text classification datasets with detailed documentation and example usage.

---

**Tip:** Start with reference #1 for foundational concepts, then #2 for practical tools, and #3 for understanding modern tokenization approaches.
