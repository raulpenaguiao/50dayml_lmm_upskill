# Day 02: Convolutional Neural Networks - Further Reading

This day introduces Convolutional Neural Networks (CNNs), which leverage spatial structure in image data to achieve superior performance compared to fully connected networks.

## References

1. **CS231n: Convolutional Neural Networks**
   - [CS231n CNN Lecture](https://cs231n.github.io/convolutional-networks/)
   - Stanford's comprehensive guide to CNNs covering convolution operations, pooling, and architecture design with clear visualizations.

2. **PyTorch CNN Tutorial**
   - [PyTorch: Training a Classifier](https://pytorch.org/tutorials/beginner/blitz/cifar10_tutorial.html)
   - Official PyTorch tutorial implementing a CNN on CIFAR-10 with step-by-step explanations.

3. **Understanding Convolution Operations**
   - [Convolution Arithmetic Paper](https://arxiv.org/abs/1603.07285)
   - Visual guide to convolution operations, padding, stride, and transposed convolutions with clear illustrations.

4. **Deep Learning Book - Convolutional Networks**
   - [Chapter 9: Convolutional Networks](https://www.deeplearningbook.org/contents/convnets.html)
   - Theoretical foundations of CNNs from the Goodfellow, Bengio, and Courville deep learning textbook.

5. **3Blue1Brown - Neural Networks Series (Part 4)**
   - [3Blue1Brown: What is a Convolutional Network?](https://www.youtube.com/watch?v=KuXjwB4LzSA&t=0s)
   - Excellent visual explanation of convolution and how filters work intuitively.

---

**Tip:** Start with reference #5 for intuitive understanding, then #1 for practical details, and #3 for mathematical rigor.
