# Day 09: Advanced CNN Architectures - Further Reading

This day studies famous CNN architectures that shaped modern computer vision: VGGNet, ResNet, and Inception. Understanding these designs is essential for building effective deep learning systems.

## References

1. **VGGNet Paper: Very Deep Convolutional Networks**
   - [Simonyan & Zisserman: VGGNet](https://arxiv.org/abs/1409.1556)
   - Influential paper showing that network depth is crucial, introducing the VGG architecture with simple 3×3 convolutions.

2. **ResNet Paper: Deep Residual Learning for Image Recognition**
   - [He et al.: ResNet](https://arxiv.org/abs/1512.03385)
   - Groundbreaking paper introducing skip connections enabling training of networks with 100+ layers via residual learning.

3. **Inception: Going Deeper with Convolutions**
   - [Szegedy et al.: GoogLeNet/Inception](https://arxiv.org/abs/1409.4842)
   - Paper introducing multi-scale feature extraction through parallel pathways and 1×1 convolutions for efficiency.

4. **Understanding Network Architecture Blocks**
   - [CS231n: CNN Architectures](https://cs231n.github.io/convolutional-networks/#architectures)
   - Stanford lecture notes explaining VGG, ResNet, and Inception architectures with implementation details and insights.

5. **Modern CNN Architecture Designs**
   - [Papers With Code: Image Classification Models](https://paperswithcode.com/area/computer-vision/image-classification)
   - Reference implementations and comparisons of state-of-the-art CNN architectures with benchmarks.

---

**Tip:** Start with reference #4 for clear explanations, then read papers #1-3 in order (VGG → ResNet → Inception) to see evolution.
