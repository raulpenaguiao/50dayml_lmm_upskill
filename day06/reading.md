# Day 06: Transfer Learning and Pretrained Models - Further Reading

This day covers transfer learningâ€”leveraging models trained on large datasets to solve new tasks efficiently. This is one of the most practical techniques in applied deep learning.

## References

1. **Transfer Learning: A Friendly Introduction**
   - [Jeremy Howard's Fast.ai Transfer Learning](https://course.fast.ai/videos/?lesson=1)
   - Practical and intuitive introduction to transfer learning concepts with real-world examples and best practices.

2. **ResNet Paper: Deep Residual Learning**
   - [He et al.: ResNet Paper](https://arxiv.org/abs/1512.03385)
   - Seminal paper introducing residual connections that enable training of very deep networks, fundamental to modern architectures.

3. **ImageNet Pretrained Models**
   - [PyTorch Pretrained Models](https://pytorch.org/vision/stable/models.html)
   - Official PyTorch documentation for accessing and using ImageNet-pretrained models (ResNet, VGG, MobileNet, etc.).

4. **Fine-tuning Strategies and Differential Learning Rates**
   - [ULMFiT Paper](https://arxiv.org/abs/1801.06146)
   - Paper introducing fine-tuning techniques and differential learning rates, showing how to efficiently adapt pretrained models.

5. **CIFAR-10 Dataset and Benchmarks**
   - [CIFAR-10 Dataset](https://www.cs.toronto.edu/~kriz/cifar.html)
   - Original dataset page with details, and papers documenting state-of-the-art results using transfer learning approaches.

---

**Tip:** Start with reference #1 for practical intuition, then #3 to load pretrained models, and #2, #4 for theoretical understanding.
