## References

1. **Convolutional Neural Networks for Sentence Classification**
   - [Kim: CNN Text Classification Paper](https://arxiv.org/abs/1408.5882)
   - Influential paper introducing CNNs for text classification showing they can be competitive with traditional NLP methods.

2. **Hierarchical Attention Networks for Document Classification**
   - [Yang et al.: Hierarchical Attention](https://arxiv.org/abs/1606.06724)
   - Paper on attention-based hierarchical models for document classification, capturing word and sentence-level importance.

3. **Text Classification with Transformer Models**
   - [Hugging Face: Text Classification Guide](https://huggingface.co/docs/transformers/tasks/sequence_classification)
   - Modern approach using pretrained transformers for text classification with practical examples and code.

4. **Handling Class Imbalance in Text Classification**
   - [Class Imbalance in NLP](https://towardsdatascience.com/handling-class-imbalance-in-nlp-a-detailed-guide-5f3d2e82e18f)
   - Practical guide to dealing with imbalanced datasets in text classification using weighted losses and sampling strategies.

5. **PyTorch Text Classification Examples**
   - [PyTorch TorchText Documentation](https://pytorch.org/text/)
   - Official library and documentation for building text classification pipelines with PyTorch.

---

**Tip:** Start with reference #1 for CNN-based approach, #3 for modern transformers, #2 for attention, and #4 for practical issues like class imbalance.
